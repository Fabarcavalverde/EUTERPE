# Uso de Inteligencia Artificial en el Proyecto

Este proyecto hizo uso responsable y limitado de herramientas de Inteligencia Artificial como apoyo complementario durante el desarrollo técnico y la documentación.

Las herramientas fueron utilizadas principalmente para:

- Resolver dudas conceptuales específicas sobre POS Tagging.
- Aclarar diferencias entre sistemas de etiquetas (Penn Treebank vs Universal POS).
- Mejorar redacción técnica del README.
- Obtener sugerencias de estructura para documentación.
- Resolver errores puntuales de configuración del entorno.

El diseño del pipeline, la implementación del código principal, el análisis de resultados y la interpretación de métricas fueron desarrollados directamente por el equipo.

# Herramientas Utilizadas

- ChatGPT (OpenAI)
- GitHub Copilot (uso ocasional para autocompletado)

# Modificaciones Realizadas

El código sugerido por IA fue:

- Ajustado a la estructura específica del proyecto.
- Simplificado para mantener coherencia con el pipeline diseñado.
- Adaptado para trabajar con el dataset utilizado.
- Revisado manualmente para asegurar comprensión completa de cada paso.

No se utilizó código generado automáticamente sin revisión o adaptación.

# Reflexión sobre el Aprendizaje

La Inteligencia Artificial fue utilizada como herramienta de apoyo puntual, similar a consultar documentación técnica o foros especializados. Su uso permitió resolver dudas específicas de manera más eficiente y mejorar la claridad en la documentación.

El proceso de diseño, implementación y análisis fue realizado conscientemente por el equipo, asegurando comprensión técnica del etiquetado morfosintáctico, las métricas lingüísticas y el flujo completo del proyecto.

El uso de IA facilitó el aprendizaje, pero no sustituyó el razonamiento ni el trabajo técnico realizado.
